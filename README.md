# Work Time Tracker Android v6

نسخه موبایل شخصی برای ثبت فعالیت برنامه‌ها، گزارش‌گیری، اسکرین‌شات اختیاری با Accessibility و به‌روزرسانی زنده.

## تغییرات v6

- اضافه شدن قفل ورود عددی با رمز پیش‌فرض `1183`؛ بعد از وارد کردن رمز و زدن `=` وارد محیط اصلی می‌شوید.
- برای شفافیت و جلوگیری از مخفی‌سازی، برنامه همچنان با نام Work Time Tracker نمایش داده می‌شود و خودش را به‌عنوان اپ دیگری جا نمی‌زند.
- اضافه شدن کنترل‌های `+` و `−` روی تایم‌لاین صفحه خانه.
- زوم تایم‌لاین بین 24، 12، 6 و 3 ساعت تغییر می‌کند و همزمان با داده‌های زنده به‌روزرسانی می‌شود.
- حفظ اصلاحات UI ریسپانسیو، فاصله امن از Status Bar / Navigation Bar، و انتقال دسترسی‌ها به صفحه تنظیمات.
- ثبت فعالیت زنده هر 1 ثانیه و نمایش رکورد در حال اجرا.

## ساخت APK در GitHub Actions

فایل‌ها را مستقیم در ریشه repo بگذارید؛ یعنی `app`، `.github`، `build.gradle` و `settings.gradle` باید مستقیم دیده شوند. سپس workflow موجود را از تب Actions اجرا کنید.

## مجوزها

- Usage Access برای ثبت فعالیت برنامه‌ها لازم است.
- Screenshot Accessibility برای ذخیره اسکرین‌شات کوچک کنار فعالیت‌ها لازم است.
- اگر Android دسترسی را با Restricted Settings رد کرد، از صفحه App info برنامه گزینه Allow restricted settings را فعال کنید.
